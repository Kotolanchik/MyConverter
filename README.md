## ConverterService
Сервис конвертирует данные из файла с расширением xml в json и наоборот.
### Входные данные
Первым аргументов входных данных будет выбор вида преобразования данных:
1. XML2JSON
2. JSON2XML

Второй и третий аргумент – это файлы, с которых происходит чтение и запись данных

Пример: XML2JSON input.xml output.json
